# InteractiveVisualization

인터렉티브 시각화는 컴퓨터그래픽스를 OpenGL을 이용하여 학습하고 충돌, 쉐이딩 ,텍스쳐를 구현합니다. 이후 C++를 활용하여 GPU를 이용한 프로그래밍과 파티클 시스템을 구현합니다.

## Overview

### 1_GPU_Drawing

OpenGL을 사용하여 기본적인 도형을 그립니다. 또한 CUDA를 이용하여 GPU 컴퓨팅의 기초 행렬연산을 진행해봅니다.
